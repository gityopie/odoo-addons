Google Maps Drawing

Allows you to draw a shape on map

Demo Video

New Widget map_drawing_shape

Enables you to draw three kind of shapes

  • Rectangle
  • Polygon
  • Circle

This module contains a demo module (custom module) of how to use the widget and also, a simple use case that used this feature
